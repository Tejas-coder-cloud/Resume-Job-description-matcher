# Field Project

A Python-based project for job analysis and embeddings.

## Project Files

- `app.py` - Main application file
- `jobs_processed.csv` - Processed jobs data
- `job_embeddings.npy` - Job embeddings generated from data
- `knowledge_base.txt` - Knowledge base reference
- `skills.txt` - Skills data

## Setup

1. Clone the repository
2. Install dependencies (if using a requirements.txt)
3. Run the application:
   ```bash
   python app.py
   ```
